## html_saver

Just a simple gem to save the content in HTML file. 

------------------------------------------------------------------------------------------------------------

## How to install?

1. Create Gemfile to your project:
```
touch Gemfile
```
2. Add to your Gemfile the following:
```
gem 'html_saver', git: 'https://github.com/KateHaman/html_saver', branch: 'main'
```
3. Run bundle:
```
bundle
```
