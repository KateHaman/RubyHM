## Pandagotchi 3.0

Now it is possible to play **your pandagotchi** in web. 
--------------------------------------------------------------------------------------------

### How to Play
Please run in the terminal:
```
ruby app.rb
```

Please open in the browser:
```
http://localhost:4567/
```

Please use the following credentials to log in.
```
username: panda
password: panda
```
