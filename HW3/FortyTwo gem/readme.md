## FortyTwo gem

This is just a simple gem which helps to get to know if the number given is 42.

------------------------------------------------------------------------------------------------------------

## How to install?

1. Create Gemfile to your project:
```
touch Gemfile
```
2. Add to your Gemfile the following:
```
gem 'FortyTwo'
```
3. Run bundle:
```
bundle
```
